> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fix 12 gaps between the SDK and the Claude Code skill, focusing on context management, bookkeeping, and judge investigation quality.

**Architecture:** Most fixes are in `refine.py` (orchestrator) and `prompts.py` (context construction). The core issue is that the skill uses the setup brief as a persistent config that enforces compliance across all iterations — we need to propagate it properly. Additionally, the skill writes trajectory.md to disk so agents can investigate it, and composes the judge board once rather than per-iteration.

**Tech Stack:** Python 3.13, `claude-agent-sdk`, `anthropic`, `pytest`, `pytest-asyncio`

    trajectory: list[IterationRecord],
    criteria_names: list[str],
    best_idx: int,
    primary: str | None,
) -> str:
    """Format the trajectory as a markdown table matching simmer-reflect/SKILL.md format."""
    # Header: | Iteration | criterion1 | criterion2 | ... | Composite | Key Change |
    # One row per iteration record
    # Footer: Best candidate: iteration N (composite: N.N/10)

def write_trajectory_md(
    trajectory: list[IterationRecord],
    criteria_names: list[str],
    best_idx: int,
    primary: str | None,
    output_dir: Path,

    """Write trajectory.md to the output directory."""
    table = format_trajectory_table(trajectory, criteria_names, best_idx, primary)
    (output_dir / "trajectory.md").write_text(f"# Simmer Trajectory\n\n{table}\n", encoding="utf-8")

```python
def condense_key_change(report: str) -> str:
    """Condense a generator report to under 60 characters for the trajectory table."""
    if not report or report == "seed":
        return report
    # Strip markdown bold markers
    text = re.sub(r'\*\*', '', report)
    # Strip common prefixes
    text = re.sub(r'^(What changed.*?:|Report:?|Summary:?)\s*', '', text, flags=re.IGNORECASE)
    # Take first sentence
    text = text.strip().split('\n')[0].split('. ')[0]
    # Truncate
    if len(text) > 57:
        text = text[:57] + "..."
    return text.strip()
```
